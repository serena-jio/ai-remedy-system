# AI客服智能补救决策系统

基于双螺旋路径模型的AI客服服务补救策略研究成果展示系统

## 项目简介

本系统是《负责任AI驱动社会救助决策转型——基于双螺旋路径模型的AI客服服务补救策略研究》的可视化展示平台，整合了15,000条真实评论数据分析和1,830份实验样本研究成果。

## 核心功能

### 1. 研究成果展示
- **非对称匹配效应**：技术性失败 × 能力导向补救效果最优
- **拟人化双刃剑效应**：高拟人化在不同失败场景下的差异化影响
- **双螺旋路径模型**：能力螺旋与温暖螺旋的心理传导机制

### 2. 智能补救决策系统
- **算法识别层**：基于TextCNN的失败类型自动识别（准确率92.3%）
- **用户分类层**：K-means聚类的四类用户画像
  - 技术理性型 (22.2%)
  - 情感敏感型 (28.1%)
  - 平衡务实型 (31.5%)
  - 低卷入型 (18.2%)
- **补救决策层**：贝叶斯网络的智能策略推荐

### 3. 在线演示系统
- 交互式补救策略推荐
- 基于失败类型、用户类型、拟人化程度的智能决策
- 实时效果预测与行动建议

## 技术栈

- **UI框架**：Tailwind CSS

## 核心研究发现

### 1. 非对称匹配修复效应
- 技术性失败 + 能力导向补救：留存意愿提升 **+85%**
- 互动性失败 + 情感导向补救：留存意愿提升 **+72%**
- 混合补救策略整体效果最佳：**6.26/7.00**

### 2. 拟人化双刃剑效应
- 技术性失败场景：高拟人化提升留存 **+17.2%**
- 互动性失败场景：高拟人化降低留存 **-12.4%**

### 3. 双螺旋路径机制
- **能力螺旋**：技术性失败 → 能力感知↓ → 算法厌恶↑ → 分配公平↓ → 留存意愿↓
  - 解释力：**78.3%**
- **温暖螺旋**：互动性失败 → 温暖感知↓ → 情感疏离↑ → 互动公平↓ → 留存意愿↓
  - 解释力：**72.5%**
